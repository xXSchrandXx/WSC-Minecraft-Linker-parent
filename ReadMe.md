# WoltLab-Minecraft (Bukkit/BungeeCord) Project
This is the project page for WSC-Minecraft-Linker.
WSC Minecraft Linker is a bridge between the [WoltLab Suite](https://www.woltlab.com/) and Minecraft servers.
